¿Qué es TypeScript?


function repeatWord(word, times) {
  var repeated = "";
  var i;
  for (i = 0; i < times; i++) {
    repeated += word;
  }
  return repeated;
}

// ¿Mamá, puedo repetir postre?

var l = repeatWord('postre', 4);

console.log(l); //postrepostrepostrepostre

TypeScript es una superclase de JavaScript

  • Añade tipado estático a JavaScript.
  • Incluye características de ECMAStript2015 (objetos basados en clases, módulos, funciones anónimas abreviadas)
  • Añade características extra como interfaces, namespaces, decorators, entre otros.
  • Es compilado a JS legible.
  • Open Source. Desarrollado por Microsoft
